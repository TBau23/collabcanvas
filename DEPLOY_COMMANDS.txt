-------------------
FIRST TIME SETUP
-------------------

1. Login to Firebase
   firebase login

2. Verify/select your project
   firebase use <your-project-id>

3. Set OpenAI API key (REQUIRED for AI agent)
   firebase functions:secrets:set OPENAI_API_KEY
   (Paste your OpenAI key when prompted)

4. Install all dependencies
   npm install
   cd functions && npm install && cd ..

-------------------
DEPLOY (Every Time)
-------------------

5. Build the React app
   npm run build

6. Deploy everything to Firebase
   firebase deploy

7. Open your deployed site
   firebase open hosting:site

-------------------
DONE! üéâ
-------------------

Your app should now be live at:
https://<your-project-id>.web.app

-------------------
TROUBLESHOOTING
-------------------

View function logs (if AI agent not working):
   firebase functions:log

Deploy only specific components:
   firebase deploy --only hosting          # Just React app
   firebase deploy --only functions        # Just AI agent
   firebase deploy --only firestore:rules  # Just Firestore rules
   firebase deploy --only database         # Just RTDB rules

Force rebuild and redeploy:
   npm run build && firebase deploy --only hosting

Check Firebase project info:
   firebase projects:list
   firebase use --list

-------------------
QUICK UPDATE WORKFLOW
-------------------

After making code changes:

If you changed React code (src/):
   npm run build && firebase deploy --only hosting

If you changed Cloud Functions (functions/):
   firebase deploy --only functions

If you changed both:
   npm run build && firebase deploy
